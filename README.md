# SOC-Home-Lab-Wazuh
A personal SOC lab using Wazuh SIEM for security monitoring, threat detection, and incident investigation.

## Overview

This project is a personal Security Operations Center (SOC) home lab built to practice security monitoring, threat detection, and incident response.

## Tools Used

- Wazuh SIEM
- Ubuntu Linux
- Windows Endpoint
- VirtualBox
- Windows Event Logs

## Lab Architecture

Windows Endpoint
        |
        |
   Wazuh Agent
        |
        |
 Wazuh Manager
        |
        |
 Wazuh Dashboard

## Objectives

- Monitor endpoint activity
- Analyze security alerts
- Detect suspicious behavior
- Practice SOC investigation workflow

## Completed Investigations

- Brute Force Login Detection
- Windows Authentication Monitoring

## Skills Practiced

- SIEM monitoring
- Log analysis
- Threat detection
- Incident investigation
- Security reporting
